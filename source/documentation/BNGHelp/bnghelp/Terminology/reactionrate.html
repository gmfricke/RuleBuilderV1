Reaction Rate


Each reaction rule is associated with a rate law, which applies to all reactions within the class of reactions defined by the rule. The reaction rate is assumed to have the form of a rate law for an elementary chemical reaction with the appropriate number of reactants, and one must be provided for each direction of a reaction rule it is used in.


Within RuleBuilder, after placing either the forward production operator or the forward and reverse production operator between species on each side of the reaction, the rate(s) are set while in object manipulation mode. Reselecting the arrow will bring up a rate dialog box that allows setting of either the forward rate only or forward and backward reaction rates.
