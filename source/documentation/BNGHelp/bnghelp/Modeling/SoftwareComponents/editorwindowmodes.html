Editor Window Modes


The tool bar at the top of the editor window describes each of the actions possible while working with objects in the editor window. The button selections from the left hand side to the Simulation Settings button puts the user into a specific mode which is shown at the bottom of the window. Each mode corresponds to a definitive action, in order to switch actions such as changing from moving an object to defining a new one, the user must explicitly change modes by selecting the appropriate tool bar button as desired.


  • Object Manipulation Mode
  • Add Component Mode
  • Add Container Mode
  • Add Edge Mode
  • Add Forward Production Operator Mode
  • Add Forward and Reverse Production Operator Mode
  • Add Plus Operator Mode
  • Model Parameters Mode
