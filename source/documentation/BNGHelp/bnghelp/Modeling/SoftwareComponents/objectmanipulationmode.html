Object Manipulation Mode




The Object Manipulation Mode is the main mode for moving objects within the Editor Window. It is used to move components, containers and both forward and forward/reverse operators physically within the Editor Window. This allows the user to specify how a particular reaction is modeled: components can be moved in and out of containers, containers may be positioned relative to each other to signify their role as either reactants or products and the And, Forward, and Forward/Reverse Production Operators may be added to signify how a reaction is to proceed.


In addition to moving within the Editor Window, this mode allows the user to drag and drop objects into the other windows such as the Molecule Types Palette and Species Palette. This action registers objects such that they can be used as building blocks in creating reaction rules. Fully specified reaction rules may also be selected and dropped into the Reaction Rules Palette by first selecting the participating objects on the Editor Window and dropping the entire set into the Reaction Rule Palette.
