Palette Windows


The palette windows serve as finished parking spaces for built-up molecules, species, observables and reaction rules. By default, the RuleBuilder interface opens the Molecule Template palette, Species and Reaction Rule palette windows. The remaining Observables and Simulation palettes are selected for use through View menu Observables Palette and Simulation Palette options.
