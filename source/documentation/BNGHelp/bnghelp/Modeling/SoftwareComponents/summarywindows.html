Summary Windows


The contextual environment windows are non-interactive displays that highlight the status of the model, network generation or simulation. They may be selected through the View menu Reaction Window, Model Summary and Debug Output Window options.


  • The Reaction Window is used to display reactions that occur as a result of the seed reaction rules in the initial network. These are automatically generated, based on model set up and parameters.
  • The Model Summary displays all user input and default parameters that are part of a model.
  • The Debug Output Window only comes into play if a bug is encountered. The contents of this window may be sent as part of a bug report.
