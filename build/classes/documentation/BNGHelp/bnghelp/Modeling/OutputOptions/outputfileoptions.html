BioNetGen Output Products


The network built by the BioNetGen Engine is output mybngmodel.xml for a model specified with the name mybngmodel. This SBML encoding can be exported to any SBML processing tool.


The timecourse simulation concentrations of all species and observables are contained in the text data files mybngmodel.cdat and mybngmodel.gdat respectively. Specific times are preset as input parameters prior to simulation execution. The timecourse results can be easily viewed with the Grace graphics package.
