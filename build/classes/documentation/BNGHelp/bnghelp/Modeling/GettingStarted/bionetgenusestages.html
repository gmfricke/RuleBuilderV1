BioNetGen Use Steps


There are five distinct steps required to fully model and simulate a biochemical network within BioNetGen. BNGL files created and saved in Steps 1 - 3 are easily reused. Selecting open from the BioNetGen File menu option will automatically read in a saved .bng file and redraw all objects within the palette windows.


  • Step 1: Specify model
  • Step 2: Set simulation parameters
  • Step 3: Save model and parameters
  • Step 4: Execute simulation
  • Step 5: View simulation results
