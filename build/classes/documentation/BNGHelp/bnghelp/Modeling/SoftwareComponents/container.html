Container


A container is used within the Editor Window to denote a molecule boundary. By clicking on the Container tool bar button a user enters Add Container Mode. Containers may be placed around existing components already present in the Editor Window to encapsulate the components, or it may be drawn first followed by adding one or more components within. A resize handle is provided to stretch the container to larger or smaller sizes as desired.


A container should be added to the Molecule Template Palette once the user is satisfied with its specification. If no states have been set for internal components the user will be prompted to add them prior to registrations into the Molecule Template.
