Add Forward and Reverse Production Operator Mode




The Add Forward and Reverse Production Operator Mode allows the user to specify a bidirectional reaction rule. This is equivalent to specifying two unidirectional reaction rules with the Forward Production Operator, hence with this mode the reaction rates must be specified for each direction. To do this the user must switch to the Object Manipulation Mode, select the Forward/Reverse Production Operator and click to bring up the edit window for the object.
