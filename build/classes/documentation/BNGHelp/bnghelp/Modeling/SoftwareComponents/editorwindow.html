Editor Window


The Editor Window serves as the central blackboard workspace. The variety of modes allow the user to build up molecules and reaction rules. After each object has been created and registered within the appropriate Palette Window, it may be cleared from the workspace by hitting the cut button or by dragging a boundary line across all window objects while in object manipulation mode and deleting.


An effective strategy for building up a model is to create the molecules one by one and register them into the Molecule Template Palette and Species Palette, clearing the workspace between each new entity. Then to create the reaction rules, the participating species may be dragged and dropped from the Species Palette onto the Editor Window to form a rule. As before it is advised to clear the Editor Window between specifying different rules. The zoom out, zoom in and autozoom buttons, or zoom slider, allow the user to specifically tailor workspace magnification.
