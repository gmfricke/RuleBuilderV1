Copy Button




The Copy Button is available on the Editor Window as well as the Palette Windows. It remembers either single or multiple objects that have the current focus. Along with the Paste Button, the Copy Button allows for single or multiple objects to be duplicated at will within a window.


Copy is meaningful only within a single window, rather than across windows. Objects selected via Object Manipulation Mode will display their focus in blue and as with the other buttons, multiple objects may have a boundary drawn around them as desired prior to a copy action.
