Cut Button




The Cut Button is available on the Editor Window as well as the Palette Windows. It clears and remembers either single or multiple objects that have the current focus. As with the Delete Button, this button allows for clean up of a window, but will work in concert with the Paste Button to allow relocation to a new location within a window.


Cut and Paste are meaningful only within a single window, rather than across windows. Object Manipulation Mode allows selection of the offending items and will display their focus in blue, multiple objects may have a boundary drawn around them as desired.
