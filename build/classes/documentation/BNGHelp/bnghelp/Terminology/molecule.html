Molecule


A molecule is comprised of multiple components. A molecule may have one or more domains, as described by its components, where each may have one or more states. Graphically this is represented by a box surrounding a set of nodes that represent each component of the molecule. This box then serves as a unit, which can be named. Molecules with no set states will display with a red outline, molecules with set states will display with a green outline.


A FcεRI receptor, bivalent IgE ligand, Src family kinase Lyn or protein tyrosine kinase Syk are all examples of molecules.
