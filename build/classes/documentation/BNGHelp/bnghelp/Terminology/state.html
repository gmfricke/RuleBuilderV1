State


A state is an attribute of a molecular component, for example attributes such as non-covalent ligation, conformation or covalent modification might be used to describe possible variations of a component. Each of these attributes comprises a single state and may influence the rate of a reaction. Components may have multiple states as well.


The user assigns states two different ways. Dragging a molecule over to the Molecule Types Palette will trigger a prompt to set node states. States may be continue to be added until the Done option is selected. Additionally a molecule already present in the Molecule Types Palette may be edited directly to add, modify or delete states as needed. Again, once the use is satisfied, the Done option should be selected. States may be set for each node of a molecule.


A molecule with at least one state is valid as indicated in the Molecule Types Palette with a green outline. Unfinished molecules with no defined states retain their red color until the minimum of one state has been set.
